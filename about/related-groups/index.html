Other Linux-related groups

PLUG is affiliated with Linux Australia , who support the Australian Linux community in a number of ways, including helping to organise linux.conf.au , Australia’s national Linux conference.

Other Linux User Groups (LUGs) around the country include:

  • Some current, some historical: Whirlpool wiki
  • ACT: Canberra Linux User Group (CLUG)
  • New South Wales: Sydney Linux Users' Group (SLUG)
  • Queensland:
  • Home Unix Machine, Brisbane User Group (HUMBUG)
  • Central Queensland Linux Users Group (CQLUG)
  • South Australia: LinuxSA
  • Tasmania: Tasmanian Linux Users' Group (TasLUG)
  • Victoria:
    • Free Software Australia, Melbourne branch (FSM)
    • Linux Users of Victoria (LUV)
    • Melbourne Linux Users' Group (MLUG)

Please contact us to have your group listed!
